 # QML_Medical

 # Вашему вниманию представлено приложение "Регистратура", которое реализует взаимодействие пользователя с базой данных SQLite, через удобный и понятный интерфейс выполненный на языке QML.

 ## Принцип работы приложения

После запуска приложения пользователь нажимает на кнопку "Подключиться к БД". Далее идет подключение к базе и индикация основных элементов представления. Кнопки становятся доступны пользователю, и на экране выводятся данные из файла БД в табличном виде, в statusbox выводится дополнительная информация. 

 ## Выбор таблицы осуществляется пользователем через comboBox обозначенный как "Выбор таблицы"

 -  Доступны таблицы "Пациенты" и  "Доктора"

## У каждого табличного представляения свои Роли
Использование ролей позволяет быстро и нативно ориентироваться в данных пациентов и докторов.

- Таблица "Пациенты". В зависимости от возраста пациента, его строка записи в таблице инициализируется соответствующим цветом. Жёлтый цвет - пациент до 10 лет,светло-зелёный цвет 10-20 лет,зелёный цвет 20-40 лет, розовый цвет 40-60 лет, красный цвет  60 и более лет.
- Таблица "Доктора". В зависимости от специализации доктора, его строка в таблице инициализируется соответствующим цветом. Жёлтый - терапевт, светло-зелёный цвет - хирург, зелёный цвет - гастроэнтеролог, фиолетовый цвет - офтальмолог,невролог - розовый цвет, красный цвет - отоларинголог.

## Добавление и удаление записей
 * При добавлении записи пользователь должен обязательно заполнить поля ввода "Имя" и "Категория". В случае с записями пациентов это имя пациента и год рождения соответственно. При работе с записями докторов указывается имя доктора и его специализация.
 * При удалении записи пользователь может ввести только имя, соответсвующее требуемой позиции в таблице.
 * В случае успешной/неудачно попытки добавления/удаления данных, в внизу экрана в statusBox будет выведена соответствующая информация.
  
 ### Взаимодействие с пользователем через кнопки

 -  "Добавить запись" создание записи в БД

 -  "Удалить запись" удаление записи из БД

 -  "Сортировать" сортировка данных представленных в таблице ( по возрастанию, по убыванию) в соответствии с тем полем сортировки, которое пользователь выбрал в combobox (Подписан "Категория").

 -  "Отмена сортировки" отмена сортировки
